> https://giaconbruno.github.io/ >>

## ------ Requisito: NODE.JS ------ 

### Acessar a pasta onde irá criar o projeto
```sh
cd "caminho da pasta"
```

### Criar projeto Vue
```sh
vue create "nome do projeto"
```

### Acessa a pasta do projeto
```sh
cd "nome do projeto"
```

### Instalar Dependências
```sh
npm i vue-router
```

### Inicia o projeto
```sh
npm run serve
```
